## Key Lessons Learned:

### Technical:
- Setup always takes longer than expected - **that's normal!**
- Backticks (`` ` ``) work with `${variables}` for string interpolation
- Single/double quotes (`'` `"`) show literal text, not variable values
- Always save files before running (Ctrl + S)
- `const` for values that won't change (preferred default)
- `let` for values that will change
- `var` should be avoided (messy scoping)
- Objects with `const` can have their properties modified
- Always commit working code to GitHub

## What I Understand Deeply Now:

### Variables:
- **`const`** = cannot reassign the variable itself
- **`let`** = can reassign the variable
- **`const` objects** = can't reassign the object, but CAN change properties inside
- Example: `const student = {name: "Lucy"}; student.name = "Lucy End";` ‚úÖ Works!

### Template Literals:
- Use backticks: `` `text here` ``
- Embed variables: `` `Hello ${name}` ``
- Better than concatenation: `"Hello " + name`
